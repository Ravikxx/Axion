What it does

Everything you need, nothing you don't

Axion brings together the tools that actually matter for a terminal AI agent.

⌨️

Terminal Agent

Full agentic loop in your terminal. Edit files, run commands, fix bugs — open source and fully extensible.

🧠

Memory Across Sessions

Automatically extracts and remembers context from each session — project name, decisions, file paths — injected into every future session.

🎤

Voice Input

Type /voice, speak, press Enter. Audio is transcribed by Whisper and sent straight to the agent.

🖥️

Computer Use

Vision + screenshot analysis lets the agent see your screen, click elements, and automate UI tasks.

💬

Discord Bot

Run the agent as a Discord bot. Queue tasks, cancel runs, and get full responses in any channel.

🌐

Chrome Extension

Read pages, click elements, fill forms, and automate the browser — directly from the agent.

🔌

MCP Servers

First-party MCP servers for Blender, plus any tool you can connect with the Model Context Protocol.

⚡

Multi-model

Claude, GPT-4o, Gemini, Groq, Ollama, or Lumen — switch with /model. Onboarding wizard handles setup.

🔗

OAuth Integrations

Connect GitHub, Google, Notion, and Slack with /oauth connect. Read issues, docs, and messages in context.

Quick start

One command to get started

Install globally from npm. The onboarding wizard runs on first launch — pick a model, enter your key, go.

# Install
$ npm install -g axion-cli

# Launch — wizard runs on first start if no key is set
$ axion

# Or skip straight to the free model
$ axion --model lumen
