What it does

Runs in your terminal. Does a lot.

Most of what you'd reach a browser for, Axion handles from the command line.

Edit files & run commands

The core agentic loop — reads your code, makes changes, runs tests. Fully open source, modify any part of it.

Memory across sessions

After each session, Axion pulls out what mattered — decisions, file paths, context — and injects it next time automatically.

Voice input

Type /voice, speak, press Enter. Whisper transcribes it and sends it straight to the agent.

See your screen

Takes a screenshot, describes what it sees, clicks elements. Useful for debugging UI or automating things you can't script.

MCP marketplace

/mcp install github — one command to add GitHub, Postgres, Notion, Playwright, and 10+ more MCP servers.

Browser control

The Chrome extension lets the agent read pages, click things, and fill forms — handy for scraping or automating web tasks.

Team accounts

Create a team, invite members, and share API keys. Usage is tracked per key so you always know who's using what.

Switch models anytime

Claude, GPT-4o, Gemini, Groq, Mistral, Ollama, or Lumen. /model <name> to switch mid-session.

GitHub, Google, Notion, Slack

/oauth connect links your accounts. The agent can pull issues, docs, and messages without you copying them in.

Quick start

Install and go

Global npm install, then run axion. A wizard runs once if you don't have a key yet.

# Install
$ npm install -g @axion-labs-ai/quark-cli

# Launch — setup wizard runs on first start
$ axion

# Skip to the free model
$ axion --model lumen
