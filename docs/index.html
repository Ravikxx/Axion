what it does

Runs in your terminal. Does a lot.

Most of what you'd reach a browser for, Axion handles from the command line. Every feature below ships in the open-source CLI.

the core loop

Edit files & run commands

Reads your code, makes changes, runs the tests. Fully open source — modify any part of it.

automatic

Memory across sessions

After each session, Axion keeps what mattered — decisions, file paths, context — and injects it next time.

/voice

Voice input

Speak, press Enter. Whisper transcribes it and sends it straight to the agent.

/computer

See your screen

Takes a screenshot, describes what it sees, clicks elements. Debug UI or automate what you can't script.

/mcp install github

MCP marketplace

One command to add GitHub, Postgres, Notion, Playwright, and 10+ more MCP servers.

chrome extension

Browser control

The agent reads pages, clicks things, and fills forms — handy for scraping and web automation.

teams

Team accounts

Create a team, invite members, share API keys. Usage is tracked per key so you know who's using what.

/model <name>

Switch models anytime

Claude, GPT, Gemini, Groq, Mistral, Ollama, or Lumen — switch mid-session without losing context.

/oauth connect

GitHub, Google, Notion, Slack

Link your accounts once. The agent pulls issues, docs, and messages without you copying them in.

quick start

Install and go

Global npm install, then run axion. A setup wizard runs once if you don't have a key yet.

# install
$ npm install -g @axion-labs-ai/quark-cli@latest

# launch — setup wizard runs on first start
$ axion

# skip straight to the free model
$ axion --model lumen
