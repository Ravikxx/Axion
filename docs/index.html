An AI agent
for your terminal

Edits files, runs commands, searches the web, controls your screen. Works with Claude, GPT-4o, Gemini, Groq, Ollama — or Lumen, for free.

Install free → GitHub
axion — ~/my-project
Features What you can do with it
1
Edit files & run commands
The core agentic loop. Reads your code, makes changes, runs tests. Fully open source — modify any part of it.
MIT
2
Memory across sessions
After each session Axion extracts what mattered — decisions, file paths, context — and injects it automatically next time.
3
Voice input
Type /voice, speak, press Enter. Whisper transcribes it and sends it to the agent.
4
Computer use
Takes a screenshot, describes what it sees, clicks elements. Useful for debugging UI or automating tasks you can't script.
5
Discord bot mode
Run Axion as a Discord bot — queue tasks, cancel runs, get full output in any channel.
6
Browser control
The Chrome extension lets the agent read pages, click things, and fill forms.
7
Multi-model
Claude, GPT-4o, Gemini, Groq, Mistral, Ollama, or Lumen. /model <name> to switch.
8
OAuth integrations
/oauth connect links GitHub, Google, Notion, and Slack. Pull issues, docs, and messages without copying them in.
9
MCP servers
Plug in any Model Context Protocol server. Blender support ships out of the box.

Quick start Install and go — wizard runs on first launch
# Install
$ npm install -g axion-cli

# Launch — setup wizard runs if no key is set
$ axion

# Or go straight to the free model
$ axion --model lumen
